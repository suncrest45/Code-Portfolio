

# [Spirit & Steel](https://store.steampowered.com/app/4426980/Spirit__Steel/)
## UI/Telemetry Programmer | 28-person team
### Responsibilities:

- General framework - I made a stack that was managed by the game instance for push and popping widgets 
- Implemented a custom navigation framework that supported mouse, keyboard, and gamepad navigation with icon switching between the major input types and different controller types (PS5/Xbox)
- Handled all major menu and button behaviours - main menu, settings menu, pop-up menus, button highlighting 
- Vertical navigation - Handled in blueprints by treating vertical/horizontal boxes as an array
- Horizontal navigation - Handled in C++ using map of function pointers
